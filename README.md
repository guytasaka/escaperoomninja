# Escape Room Ninja

Monorepo scaffold for Escape Room Forge using Turborepo + pnpm.

## Quick start

```bash
pnpm install
docker compose up -d
cp .env.example .env
pnpm dev
```

## Common commands

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```
